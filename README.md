## Atualizando

Tem script "update-ivertion-chat" para facilitar atualização ele somente funciona se tiver instalado pasta ivertion.chat.io

sh update-ivertion-chat

<br/>

**IMPORTANTE**: verifique sempre o .env.example e ajuste o seu .env antes de atualizar, uma vez que algumas novas variáveis podem ser adicionadas.
